Bank Management System

A desktop-based Java application that allows users to perform core banking operations such as deposit, withdrawal, fund transfer, and balance inquiry. Data is stored securely in a MySQL database with Java Swing UI.

Features:

  • 5-button UI: Withdraw, Deposit, Transfer, Balance, Exit
  • Object-Oriented Design
  • Secure MySQL integration
