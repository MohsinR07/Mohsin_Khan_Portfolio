Project information

  • Client: Ideal Public School

Overview

A class-wise School Management System featuring Admin, Teacher, and Student panels. Designed for Ideal Public School, this system tracks attendance, results, teachers, and more.

  • 🧑‍🏫 Admin Panel: Manage students, teachers, attendance, results
  • 📘 Teacher Panel: Add marks, daily attendance, mark own attendance
  • 🎓 Student Panel: View attendance, marks, basic profile
  • 📅 Month-wise filter and report generation
  • 📱 Fully responsive & mobile friendly
